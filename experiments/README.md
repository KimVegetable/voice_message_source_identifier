# Revision Experiments

This directory contains manuscript revision experiments that are separate from the released source identification tool.

- `sota_baselines`: paper based Jin et al. and Wang et al. comparison code, with the curated result tables used for the SOTA comparison.
- `current_version_lr`: current app version validation scripts for the logistic regression component used in the revised manuscript analysis.

Generated caches and rerun outputs are written to local `results` directories and are ignored by Git. Curated paper result snapshots are kept in `paper_results`.
